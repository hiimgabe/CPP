# CPP

This repository contains the C++ modules implemented during the common core of 42 school. Each module covers various topics and exercises designed to enhance our understanding of C++ programming concepts.

Below is a summary of each module along with links to their respective repositories:

| Module | Description |
| ------ | ----------- |
| [CPP00](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-00) | Namespaces, classes, member functions, stdio streams, initialization lists, static, const, and some other basic stuff. |
| [CPP01](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-01) | Memory allocation, pointers to members, references and switch statements. |
| [CPP02](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-02) | Ad-hoc polymorphism, operator overloading and the Orthodox Canonical class form. |
| [CPP03](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-03) | Inheritance, and virtual inheritance.  |
| [CPP04](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-04) | Subtype Polymorphism, Abstract Classes, and Interfaces. |
| [CPP05](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-05) | Exceptions, pointers to member function and more. |
| [CPP06](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-06) | Type casting. |
| [CPP07](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-07) | C++ templates. |
| [CPP08](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-08) | Standard Template Library (STL), Templated containers, iterators, algorithms. |
| [CPP09](https://github.com/hiimgabe/CPP/tree/main/CPP-Module-09) | More STL iterators, and containers. |
