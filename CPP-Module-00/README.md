# CPP00: Introduction
This module is the introduction of the 42 school C++ modules. The objective of this module is to get acquinted with the following notions: Namespaces, classes, member functions, stdio streams, initialization lists, static, const and other basic concepts. See [subject](en.subject.pdf).

### Exercise00: Megaphone
Implement a simple program that prints the message passed as argument in upper case letters.

### Exercise01: My Awesome PhoneBook
Implement an interface for a phonebook in which contact info is stored and can be later retrieved with the help of implemented commands: ADD, SEARCH and EXIT. Stores up to 8 contacts.

### Exercise02: The Job Of Your Dreams
In this exercice, the student is asked to implement the functions for the prototypes contained in the .hpp file coming with the subject so the tests contained in tests.cpp compile successfully.
